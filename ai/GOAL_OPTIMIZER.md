
# Goal Optimization Engine

Purpose:
Compute optimal savings plans for financial goals.

Directory:
backend/app/goals/

Example goals:

emergency_fund
buy_house
education_fund
retirement

Optimization problem:

Minimize:
time_to_goal

Subject to:

income
expenses
debt_payments

Example output:

{
 "goal": "emergency_fund",
 "recommended_saving_per_month": 4000000,
 "completion_time": "18 months"
}
